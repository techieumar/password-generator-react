Password Generator

A modern and responsive Password Generator Web App built using React.js and Tailwind CSS. It allows users to generate secure passwords with customizable options.

🚀 Features
🔢 Adjustable password length (8–50)
🔤 Include/exclude numbers
🔣 Include/exclude special characters
📋 Copy to clipboard functionality
🌙 Dark / ☀️ Light theme toggle
⚡ Instant password generation
📱 Fully responsive design
🛠️ Tech Stack
React.js
JavaScript (ES6+)
Tailwind CSS
Vite

📂 Project Setup
1. Clone the repository
git clone https://github.com/techieumar/password-generator-react.git
2. Navigate to project folder
cd password-generator-react
3. Install dependencies
npm install
4. Run locally
npm run dev
